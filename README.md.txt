 📊 Python Web Scraper – Quotes Data Extractor

 📌 Overview

This project is a Python-based web scraper that extracts quotes, authors, and tags from a website and stores the data in a well-structured Excel file.

It demonstrates practical use of web scraping, data processing, and automation using Python.

---

🚀 Features

* Scrapes quote text, author, and tags
* Supports multi-page scraping (pagination)
* Stores structured data using pandas
* Exports data to Excel (.xlsx)
* Clean and readable formatted output

---

🛠️ Technologies Used

* Python
* requests
* BeautifulSoup
* pandas
* openpyxl

---

📂 Project Structure

project-folder/

⚙️ How to Run

```

cd your-repo-name
```

3. Install dependencies:
pip install -r requirements.txt
```

4. Run the script:
python scraper.py
```

 📊 Output

The script generates an Excel file (`quotes.xlsx`) containing:

* Quote text
* Author name
* Tags
